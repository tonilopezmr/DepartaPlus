DEPARTAPLUS+ 
==================

	DepartaPlus es una pequeña aplicacion que da de alta, baja, modifica y consulta empleados
	y departamentos.


DEPARTAPLUS+ 1.2.0
==================
	
	Version con JDBC, con los patrones de diseño DAO y Vo, directamente programando las
	sentencias SQL.


DEPARTAPLUS+ 2.0.1 ORM
==================

	Version con el framework Hibernate, que fue directamente actualizado de la version
	anterior.
